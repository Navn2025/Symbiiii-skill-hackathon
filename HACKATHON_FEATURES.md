## ğŸ” Security Checklist

- [ ] **Authentication**: JWT with expiration, refresh tokens
- [ ] **Authorization**: Role-based access control (RBAC)
- [ ] **Input Validation**: Sanitize all user inputs
- [ ] **SQL Injection**: Use parameterized queries/ORMs
- [ ] **XSS Prevention**: Escape HTML, use CSP headers
- [ ] **CSRF Protection**: CSRF tokens for state-changing ops
- [ ] **Rate Limiting**: Prevent API abuse (100 req/min)
- [ ] **Password Security**: bcrypt with salt rounds 10+
- [ ] **HTTPS**: SSL/TLS certificates (Let's Encrypt)
- [ ] **Data Encryption**: Encrypt sensitive data at rest
- [ ] **Session Management**: Secure cookies, httpOnly flag
- [ ] **Dependency Scanning**: No known vulnerabilities
- [ ] **CORS**: Properly configured origins
- [ ] **Error Handling**: Don't leak stack traces to users
- [ ] **Logging**: Log security events, detect anomalies
